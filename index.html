This tool presents validated vector options for the most common model organisms. For other species, any published results can be found in the Caltech CLOVER Center's Systemic AAV Repository.





The enhanced properties of some rodent vectors depend on the strain used. For more details on this phenomenon, please see: Addgene Blog: Systemic Capsids section 'Strain-specific tropism in mice.'





Vectors have enhanced performance in the below tissues. Most commonly, this is layered atop AAV9's basal peripheral tissue preferences.





Broad indicates vector transduction across major cell types (e.g. neurons, astrocytes, etc) or, when following a specified cell type, across subtypes.
